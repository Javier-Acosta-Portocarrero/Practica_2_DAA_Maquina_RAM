# Práctica 2 – Máquina RAM

## Descripción general

Se ha implementado una **máquina RAM** siguiendo las instrucciones proporcionadas en el enunciado de la práctica, los **principios SOLID** y el **patrón estrategia**. El programa permite analizar la complejidad de los programas RAM introducidos a través de la cantidad de instrucciones ejecutadas para cada uno, además imprime los errores producidos si el programa no es válido junto con la línea que provocó el error.

## Requisitos

Compilador: g++
Herramienta de compilación: make

## Compilación

Desde el directorio raíz del proyecto, ejecutar:

make

Esto compilará el código fuente y generará el ejecutable principal.

## Ejecución

Una vez compilado el proyecto, ejecutar:

./practica2_DAA

El programa mostrará información de cómo ejecutarlo y qué parámetros pasar.

## Limpieza del proyecto

Para eliminar los archivos generados durante la compilación ejecutar:

make clean
